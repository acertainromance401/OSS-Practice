【기본 정보 (Basic Info)】

프로젝트명:
  SentiVision(센티비젼)

기여자:
  임재현

설명:
  사용자가 그린 그림의 색감을 분석하여 사용자의 감정 상태를 분석하는 프로그램

동기 (Why):
  행동 제어나 목적에 맞는 작업을 해주는 인공지능은 많지만, 아직 사람의 감정을 파악하는
  감성 컴퓨팅 분야는 개발이 미비함. 색감 분석을 시작점으로 하여 사람의 감정 표현을
  이해하는 감성 컴퓨팅 분야의 발전을 도모하기 위해 이 프로젝트를 시작했음.

예상 결과:
  1. iOS 앱: 사용자가 캔버스에 그림을 그릴 수 있는 드로잉 인터페이스 실현
  2. 감정 분석 엔진: RGB 색감 → 감정 분류 모델(정확도 85% 이상)
  3. 분석 결과 리포트: 감정 분류 결과 시각화 및 색상-감정 매핑 결과 표시
  4. 피드백 시스템: 사용자 수정 데이터 자동 수집으로 모델 지속 개선
  5. 감정 데이터베이스: 100+ 색상-감정 쌍으로 확장된 학습 데이터셋

【기술 스택 (Tech Stack)】

Frontend (프론트엔드):
  Swift, SwiftUI (iOS 앱), Canvas API (드로잉 기능)

Backend (백엔드):
  Python, Flask/FastAPI (API 서버)

ML/AI 라이브러리:
  OpenCV, scikit-learn (색감 분석 및 감정 분류), NumPy, Pandas

Database (데이터베이스):
  PostgreSQL (사용자 정보, 분석 기록 저장)

Deployment (배포):
  App Store, TestFlight (iOS 앱 배포), AWS (백엔드)

【주요 기능 (Key Features)】

핵심 기능 1: 현저성 맵 기반 주요 영역 감지
  - Saliency Map을 사용하여 이미지에서 시각적으로 중요한 영역 추출
  - Laplacian 엣지 감지로 사용자의 의도가 담긴 부분 강조
  - 배경 노이즈 제거 및 주요 색상에만 집중하는 전처리 과정

핵심 기능 2: 색감 추출 및 감정 분류 (KMeans + KNN)
  - KMeans 클러스터링으로 이미지에서 3가지 주도 색상 추출
  - KNN 분류기로 각 색상의 감정 예측 (RGB ↔ 감정 매핑)
  - 색상-감정 데이터베이스를 통한 실시간 감정 분석

핵심 기능 3: 사용자 피드백 기반 모델 학습
  - 사용자가 예측된 감정의 정확성 확인 및 수정
  - 피드백 데이터를 CSV에 자동 저장
  - 지속적인 학습으로 모델 정확도 향상 (Interactive Learning)
